# Reforge | Lobbysystem

## Description

*A Bukkit Plugin for 1.12.2 for Lobby*

## Features

* Maintenance
  * For Updates | ...


* ItemManager
  * Custom Items [```ItemFlag``` | ```Enchantment``` | ```Name```]


* InventoryManager
  * Load / Save Inventorys


## Getting Started

### Installing

  * Get Repository

  ```git clone https://github.com/mstvb/reforgelobbysystem.git```


### Commands

  ```/maintenance``` > Maintenance On / Off
